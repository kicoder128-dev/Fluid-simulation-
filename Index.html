Later levels require mixing colors (blue + red = purple) using special components. Controls: · Click the grid to place your selected tool · Keys 1-4 switch tools, D for delete · Space to run/stop the simulation · R to reset the level Components: · Tube - Basic pipe, connects to adjacent components · Toggle Valve - Opens/closes flow (click during simulation to toggle) · Mixer - Combines two input colors into their mixture (blue+red=purple) · Delay Tank - Stores fluid temporarily, releases when full · Input (diamond) - Source of colored fluid · Output (circle) - Target destination The simulation propagates fluid through connected components. Intermittent inputs pulse on/off, requiring timing logic. Try Level 4 for a real challenge where you must coordinate an intermittent red input with a continuous blue to create purple only when both are present.
